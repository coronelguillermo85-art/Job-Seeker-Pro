# 🕹️ Job Seeker Pro & Reclutator: The Recruitment Game

¡Bienvenido al repositorio oficial! Este proyecto no es solo un videojuego, es mi **currículum interactivo**. Diseñé esta experiencia para demostrar que en el mundo del **Management BPO y el Soporte IT**, la innovación y la resolución de problemas son la clave para superar cualquier obstáculo.

## 🎮 De qué se trata
En un mercado laboral saturado de procesos automáticos, construí este **Beat 'em up** donde el jugador (mi avatar) debe enfrentar y derrotar los desafíos reales de una búsqueda laboral:
* **ATS Destructor:** Superá los filtros automáticos con ráfagas de CVs.
* **LinkedIn Troll:** Esquivá la negatividad y el ruido de las redes.
* **El Reclutador (Boss Final):** Superá la etapa de entrevistas y "excusas modo turbo" para ganar la posición.

## 🛠️ Especificaciones Técnicas
Para este desarrollo prioricé la **eficiencia y la compatibilidad**, valores que aplico en mi gestión diaria como Team Leader:
* **Motor:** JavaScript puro (Vanilla JS) y HTML5 Canvas. Sin librerías pesadas para asegurar carga instantánea.
* **Arquitectura:** Lógica de colisiones, manejo de estados y sistema de partículas desarrollados desde cero.
* **Responsive:** Controles táctiles integrados para jugar desde el celular (ideal para reclutadores en movimiento).

## 📊 Sobre el Autor (Guillermo Coronel)
Soy **Manager BPO & Team Leader** con 9 años de experiencia liderando equipos de gran escala (+35 personas). 
* **Mi enfoque:** Unir la operatividad técnica con el liderazgo humano.
* **El objetivo de este juego:** Demostrar que tengo la infraestructura técnica y la creatividad necesaria para liderar los desafíos de 2026.
